Your agrinautical dreams have finally been answered - this is the definitive game for enthusiasts of high-tech farming.

The goal is to plow a field - but not using any boring, outdated technology such as tractors. This is the 21st century - use a rocket!
Collect cows to replenish your fuel. 

Currently, the game is early in development, with the basic rocket physics being implemented. Pretty much everything is subject to change.

Additional creatures are planned as well, each with unique effects!
